An event-driven backtester written with GUI in C++.

To build and run the project:
1. Clone the repository.
2. Create a build directory and cd into it.
(The following commands should be executed in the build directory)
3. Run `cmake ..` to generate the build files.
4. Run `cmake --build .` to build the project.
5. Run `./backtester-cpp` to execute the application. (for linux)
